Welcome to AngularKeycloakWeb!

This app demonstrates Keycloak authentication with Angular.
To use the app:

  1. Log in using the login button in the header.
  2. Once logged in, the User Profile link will appear in the navigation menu.
  3. Click User Profile to view your profile, roles, and token information.
  4. You can log out at any time using the logout button in the header.

If you are not logged in, the User Profile link will be hidden and you cannot access profile details.
